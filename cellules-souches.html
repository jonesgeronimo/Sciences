La cellule souche

Clique sur chaque cellule pour explorer sa spécialisation

Cellule souche Clique pour commencer
Elle n'est pas encore spécialisée — elle pourrait devenir presque n'importe quelle cellule du corps.
Ectoderme Clique pour explorer
Formera les couches externes : la peau et le système nerveux.
Cellule de la peau
Protège le corps contre l'environnement extérieur.
Neurone
Transmet des signaux dans le cerveau et le corps.
Mésoderme Clique pour explorer
Formera le soutien et le mouvement du corps : les muscles, le sang, les os.
Cellule musculaire
Permet le mouvement du corps.
Globule rouge
Transporte l'oxygène dans le sang.
Cellule osseuse
Donne sa structure au corps.
Endoderme Clique pour explorer
Formera les organes internes : les poumons, l'intestin, le foie.
Cellule pulmonaire
Échange l'oxygène entre l'air et le sang.
Cellule hépatique
Filtre les toxines dans le foie.
Cellule intestinale
Absorbe les nutriments provenant des aliments.
Lignée germinale Clique pour explorer
Formera les cellules reproductrices : les spermatozoïdes et les ovules.
Spermatozoïde
Transporte l'information génétique vers l'ovule.
Ovule
S'unit au spermatozoïde pour former une nouvelle vie.
